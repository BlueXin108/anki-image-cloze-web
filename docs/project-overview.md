# 项目总览

## 这个项目是什么

这个项目是一个面向 Anki 图像挖空的工作台。它的目标是把知识截图逐步变成可复习的卡片，并在自动化与人工审核之间取得平衡。

## 主要使用路径

### 流水线模式

适合希望批量处理、追求整体效率的场景。

典型流程：

1. 扫描目录
2. 运行第一阶段
3. 确认归档位置
4. 运行第二阶段
5. 确认遮罩结果
6. 导出

### 手动模式

适合希望直接控制图片、裁切和遮罩的场景。

典型流程：

1. 选择图片
2. 调整裁切范围
3. 手动画遮罩或编辑遮罩
4. 需要时把多个遮罩块合并成同一张卡
5. 预览最终结果
6. 导出

## 核心概念

- `draft`
  - 一张图片在工作台中的当前工作状态
- `mask`
  - 一个遮罩矩形块
- `card group`
  - 一个或多个遮罩块组成的一张最终卡片
- `route`
  - 目标 deck 与 tags

## 当前范围

当前这版本地优先原型已经覆盖：

- 目录扫描
- OCR
- 归档建议
- LLM 遮罩建议
- 手动编辑
- 预览渲染
- Anki 导出

## 后续方向

当前产品仍以本地优先为主，但文档和结构应尽量保持清晰，以支持后续拆出基于 Web / Server 的独立版本。
