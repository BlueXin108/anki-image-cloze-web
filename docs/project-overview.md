# 项目总览

## 这个项目是什么

这个项目当前是一条面向 Web / Vercel 的 Anki 图像挖空工作台分线。它的目标是让用户在网页里直接完成图片导入、手动遮挡、即时预览、逐张确认牌组，并最终导出到本机 Anki。

## 主要使用路径

### 手动模式

当前的主使用路径，适合希望直接控制图片、裁切和遮罩的场景。

典型流程：

1. 上传图片或导入文件夹
2. 调整裁切范围
3. 手动画遮罩或编辑遮罩
4. 需要时把多个遮罩块合并成同一张卡
5. 通过右下角导出入口进入全屏导出流程
6. 逐张确认牌组与标签，并查看只读预览
7. 在最后一步统一调整导出质量并导出到本机 Anki

### 流水线模式

当前保留为 UI 占位，用来给后续 OCR、自动归类或自动遮罩能力留接口位置。

## 核心概念

- `draft`
  - 一张图片在工作台中的当前工作状态
- `mask`
  - 一个遮罩矩形块
- `card group`
  - 一个或多个遮罩块组成的一张最终卡片
- `deck`
  - 当前图片准备导入到哪个 Anki 牌组

## 当前范围

当前这版网页工作台已经覆盖：

- 多图上传
- 文件夹导入
- 浏览器本地保存
- 手动编辑
- 即时预览
- 全屏逐张确认导出流
- 直接连接 AnkiConnect 导入

## 后续方向

后续方向是：

- 在不破坏当前手动网页链路的前提下
- 逐步把 OCR、自动归类、自动遮罩的一部分能力接回网页版本
